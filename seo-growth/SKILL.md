# SEO & Growth

## Técnico
- Title/meta description únicos por página
- Canonical URLs, sitemap.xml, robots.txt
- OpenGraph + Twitter Card meta tags
- schema.org JSON-LD: FAQ, Product, Organization
- H1 único por página, jerarquía de headings

## Performance
- Core Web Vitals: LCP<2.5s, FID<100ms, CLS<0.1
- Minimize JS (<50KB landing pages)
- Images: WebP/AVIF, lazy-load, explicit dimensions
